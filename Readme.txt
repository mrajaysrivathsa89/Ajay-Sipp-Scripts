"Callee.sh" is a Shell Script that "REGISTER"s UAS SIP endpoint and listens to "INVITE" and "Caller.sh" Dials in and 1-1 calls is established 
 and terminated.
"Caller.sh" is a shell script that "REGISTER"s UAC SIP endpoint and Initiates 1-1 Call.
 Preconditoins: Make sure the Endpoints are created on a SIP server
 Make sure All files are kept in a single directory before starting the script
 SIP messages sample is also provided for reference on one of the endpoints
